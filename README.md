# Task Manager - WebSocket & GraphQL

**Современный менеджер задач с реальным временем обновления и космическим дизайном**

##  Особенности

### **Визуальные**
- **Анимированные космонавты** - Плавные перевороты и движение в невесомости
- **Система частиц** - Интерактивный фон с подключениями
- **Тёмная тема** - Современный градиентный дизайн
- **Адаптивный интерфейс** - Оптимизирован для всех устройств

###  **Функциональные**
-  **Безопасная аутентификация** - JWT + HTTP-only cookies
-  **Real-time обновления** - WebSocket для мгновенных изменений
-  **Три панели задач** - Все/Активные/Завершённые
-  **Умные даты** - Установка сроков выполнения
-  **Вложения файлов** - Поддержка загрузки файлов
-  **Быстрое управление** - Переключение статуса и удаление

###  **Технические**
- **GraphQL API** - Единая точка входа для запросов
- **RESTful эндпоинты** - Совместимость с традиционными клиентами
- **Двойная аутентификация** - Cookie-based + Token-based
- **Оптимизированная архитектура** - Модульная структура проекта

## Предварительные требования
- Node.js 18.0 или выше
- npm 8.0 или выше
